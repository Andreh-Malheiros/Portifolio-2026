# Portfólio — Andreh Malheiros

Portfólio profissional desenvolvido para apresentar minha atuação em **desenvolvimento web, e-commerce, sistemas e tecnologia aplicada a projetos reais**.

Este projeto não foi criado apenas como uma página pessoal. Ele funciona como uma vitrine objetiva da minha trajetória, reunindo posicionamento profissional, experiências, projetos, competências e canais de contato em uma estrutura simples, direta e responsiva.

---

## Visão geral

A proposta deste portfólio é comunicar com clareza quem eu sou profissionalmente, quais áreas fazem parte da minha atuação e como aplico tecnologia em diferentes contextos.

Atualmente, minha atuação passa por frentes como:

* desenvolvimento web;
* e-commerce;
* Shopify;
* landing pages;
* testes de sistemas;
* implantação de soluções;
* documentação técnica;
* suporte técnico;
* projetos ligados a RFID;
* uso de IA como apoio à produtividade e análise.

O objetivo é mostrar que tecnologia não se resume a ferramenta, código ou layout. Em projetos reais, ela precisa resolver problemas, organizar processos, reduzir dúvidas e funcionar fora do ambiente ideal.

---

## Objetivo do projeto

Este portfólio foi desenvolvido com foco em três pontos principais:

1. **Apresentar meu posicionamento profissional**
   Desenvolvimento web, e-commerce e sistemas, com uma abordagem prática e orientada a projetos reais.

2. **Centralizar projetos e experiências relevantes**
   Incluindo trabalhos em lojas virtuais, landing pages, sistemas, testes e implantação de tecnologia.

3. **Criar uma presença digital mais clara e profissional**
   Servindo como apoio ao LinkedIn, GitHub e futuras oportunidades profissionais ou comerciais.

## Como executar localmente

Como o projeto é composto por arquivos estáticos, não é necessário instalar dependências.

Basta clonar o repositório e abrir o arquivo `index.html` no navegador.

```bash
git clone https://github.com/Andreh-Malheiros/Portifolio-2026.git
```

Depois, acesse a pasta do projeto e abra:

```bash
index.html
```

Também é possível utilizar uma extensão como **Live Server** no VS Code para visualizar o projeto localmente com recarregamento automático.
